# Avantizon Case Study

## Project Overview

Avantizon is a digital marketplace platform designed to connect users with freelancers and small-to-medium businesses offering both online and in-person services. The platform also includes a community section and an educational hub containing case studies to support users, service providers, and businesses.

This public repository is a case study documenting my development contribution to Avantizon. The main source code is not included because the project is managed through a private organisation repository.

## My Role

I worked on Avantizon as Co-founder / CTO, contributing to the planning, development, and technical direction of the platform.

My responsibilities included:

- Developing the frontend using React
- Structuring reusable UI components and platform pages
- Supporting Firebase integration for authentication, users, profiles, and marketplace data
- Working on service listings, user profiles, community features, and educational content areas
- Supporting payment-related workflows and backend planning
- Managing development through GitHub, branches, commits, and pull requests
- Using AI-assisted tools such as ChatGPT and Claude AI to support debugging, documentation, and development planning

## Key Features Worked On

### Marketplace Functionality

Avantizon was designed to support a marketplace structure where users could discover and connect with service providers. This included both online and in-person services.

### User and Provider Profiles

The platform included profile-related functionality for freelancers, businesses, and general users. This supported user identity, service information, and platform interaction.

### Community Section

A community area was included to help users engage with discussions, posts, and shared knowledge within the platform.

### Educational Hub

Avantizon also included an educational section containing case studies and learning resources to help users make better decisions when using the platform.

### Platform Development and Debugging

I contributed to debugging and improving the platform during development, including resolving issues related to React components, Firebase permissions, routing, image uploads, and data handling.
